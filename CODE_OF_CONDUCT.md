# 行为准则 / Code of Conduct

TinyML_SOC 社区中的技术讨论应保持尊重、具体并以改进工程为目标。

## 期望行为

- 按技术事实评审设计和代码。
- 报告问题时提供可复现证据。
- 明确区分实测、推断和未知项。
- 尊重许可证、署名、隐私和负责任披露要求。
- 接受纠正并保持专业讨论。

## 不可接受行为

不接受骚扰、歧视、个人攻击、蓄意干扰、发布私人信息或明知不实的技术结论。

## 执行

通过仓库维护渠道私下报告行为问题。维护者可以删除内容、关闭讨论或限制参与，并应按影响程度一致处理。

## English Summary

Keep technical collaboration respectful, evidence-based, privacy-preserving, and focused on improving the project.
